# Murth

An amusing toy programming language.


## Building

$ lein uberjar

## Usage

$ murth
Options:
         -h, --help          Display this help.
         -i, --repl          Start a repl.
         -f, --file <file>   Run source file <file>.

Example:

$ murth -f test/murth/test2.murth
Args: ("-f" "test/murth/test2.murth")
true
